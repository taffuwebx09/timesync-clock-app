# ⏰ Clock & Alarm App

A modern and interactive **Clock + Alarm Web App** built using **HTML, CSS, and JavaScript**.
This project displays real-time clock functionality along with an alarm system and theme switching.

## ✨ Features

* 🕒 Real-time Analog Clock
* 🧮 Digital Time Display
* 📅 Current Date Display
* 🌗 Light / Dark Mode Toggle
* ⏰ Set Alarm Functionality
* 🔔 Alarm Ringing Screen
* 🎯 Interactive UI with smooth design
